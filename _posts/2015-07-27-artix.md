#### 1. Connect to wifi
<br>

```
rfkill unblock wifi
connmanctl
scan wifi
services 
agent on
connect <code starting in "wifi_">
quit
```

#### 2. Partition with fdisk
<br>
sda1: 1GB
sda2: 30GB
sda3: rest

#### 3. File systems
<br>

```
mkfs.fat -F32 /dev/sda1
mkfs.ext4 /dev/sda2
mkfs.ext4 /dev/sda3
```
